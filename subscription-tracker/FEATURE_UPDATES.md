# 功能更新说明

## 已完成的更新

### 1. ✅ 删除主题颜色设置

- 从设置页面移除了主题颜色选择器
- 简化了外观设置部分

### 2. ✅ 添加语言设置

- 在设置 > 外观中添加了语言选择器
- 支持三种语言：
  - 简体中文 (zh-Hans)
  - English (en)
  - 日本語 (ja)
- 语言切换会保存到 UserDefaults
- 需要重启应用才能完全生效

### 3. ✅ 实现深色模式切换

- 深色模式设置现在可以实时生效
- 三种模式：
  - 跟随系统
  - 浅色模式
  - 深色模式
- 使用 `AppSettings` 类管理全局颜色方案
- 设置保存到 UserDefaults 和 SwiftData

### 4. ✅ 分类管理作为独立Tab

- 将分类管理从设置页面移到底部Tab栏
- 新的Tab顺序：
  1. Dashboard
  2. 订阅
  3. 分类 (新增)
  4. 设置

## 分类管理功能

### 功能列表

- ✅ 查看所有分类
- ✅ 显示每个分类的订阅数量
- ✅ 添加新分类
- ✅ 编辑分类（点击分类行）
- ✅ 删除分类（左滑）
- ✅ 颜色选择器
- ✅ 免费用户限制检查
- ✅ 空状态提示

### 分类信息

每个分类包含：

- 名称（必填）
- 描述（可选）
- 颜色（12种预设颜色）
- 关联的订阅数量

### 操作流程

#### 添加分类

1. 点击右上角"+"按钮
2. 输入分类名称
3. 输入描述（可选）
4. 选择颜色
5. 点击"保存"

#### 编辑分类

1. 点击分类行
2. 修改信息
3. 点击"保存"

#### 删除分类

1. 左滑分类行
2. 点击"删除"
3. 确认删除
4. 关联的订阅将变为"无分类"

## 技术实现

### 文件结构

```
subscription-tracker/
├── Views/
│   ├── CategoryManagementView.swift    # 分类管理主视图
│   ├── SettingsView.swift              # 更新后的设置页面
│   └── ContentView.swift               # 更新后的Tab结构
├── ViewModels/
│   └── SettingsViewModel.swift         # 添加语言设置支持
└── subscription_trackerApp.swift       # 添加AppSettings支持
```

### 核心组件

#### AppSettings

```swift
class AppSettings: ObservableObject {
    @Published var colorScheme: ColorScheme?

    func updateColorScheme(_ darkMode: Bool?)
    func loadColorScheme()
}
```

#### CategoryManagementView

- 使用 `@Query` 自动获取分类列表
- 使用 `CategoryViewModel` 处理业务逻辑
- 支持添加、编辑、删除操作
- 集成 Toast 提示和确认对话框

#### AddEditCategoryView

- 表单式编辑界面
- 颜色选择器集成
- 免费用户限制检查
- 自动保存到 SwiftData

### 深色模式实现

1. `AppSettings` 管理全局颜色方案
2. 设置保存到 UserDefaults
3. 使用 `.preferredColorScheme()` 应用到整个应用
4. 实时切换，无需重启

### 语言切换实现

1. 使用 `UserDefaults.standard.set([languageCode], forKey: "AppleLanguages")`
2. 保存用户选择的语言
3. 需要重启应用才能完全生效
4. 所有文本已支持多语言（之前已实现）

## 使用说明

### 深色模式切换

1. 打开"设置"
2. 在"外观"部分选择"深色模式"
3. 选择：跟随系统 / 浅色 / 深色
4. 立即生效

### 语言切换

1. 打开"设置"
2. 在"外观"部分选择"语言"
3. 选择：简体中文 / English / 日本語
4. 重启应用生效

### 分类管理

1. 点击底部"分类"Tab
2. 查看所有分类
3. 点击"+"添加新分类
4. 点击分类行编辑
5. 左滑删除分类

## 注意事项

1. **语言切换**
   - 需要完全关闭并重新打开应用
   - 系统级别的语言切换

2. **深色模式**
   - 立即生效，无需重启
   - 设置会保存并在下次启动时恢复

3. **分类删除**
   - 删除分类不会删除订阅
   - 关联的订阅会变为"无分类"
   - 操作不可撤销

4. **免费用户限制**
   - 最多3个分类
   - 达到限制会显示付费墙
