# 订阅详情页功能说明

## 已实现的功能

### 1. 订阅详情页 (SubscriptionDetailView)

#### 查看模式

- ✅ 显示订阅的完整信息
  - 基本信息（名称、描述、分类）
  - 计费信息（金额、周期、首次付款日期、下次续费日期）
  - 统计信息（已支付次数、历史总支出）
  - 提醒设置
- ✅ 导航栏右上角显示"编辑"按钮
- ✅ 导航栏左上角显示返回按钮（系统默认）
- ✅ 底部显示"归档"和"删除"操作按钮

#### 编辑模式

- ✅ 点击"编辑"按钮进入编辑模式
- ✅ 导航栏左上角显示"取消"按钮
- ✅ 导航栏右上角显示"保存"按钮
- ✅ 显示表单编辑界面
  - 基本信息编辑
  - 分类选择
  - 计费信息编辑
  - 提醒设置编辑
- ✅ 点击"取消"放弃更改并返回查看模式
- ✅ 点击"保存"保存更改并返回查看模式

### 2. 导航集成

#### 订阅列表页 (SubscriptionListView)

- ✅ 点击订阅卡片导航到详情页
- ✅ 左滑显示"归档"和"删除"操作（移除了"编辑"，因为在详情页编辑）

#### Dashboard 页

- ✅ 点击"即将到期"列表中的订阅卡片导航到详情页

### 3. 操作功能

- ✅ 编辑订阅信息
- ✅ 归档/取消归档订阅
- ✅ 删除订阅（带确认对话框）
- ✅ Toast 提示消息
- ✅ 自动刷新数据

## 使用流程

### 查看订阅详情

1. 在订阅列表或 Dashboard 中点击订阅卡片
2. 进入详情页查看完整信息

### 编辑订阅

1. 在详情页点击右上角"编辑"按钮
2. 修改需要更改的信息
3. 点击右上角"保存"按钮保存更改
4. 或点击左上角"取消"按钮放弃更改

### 归档订阅

1. 在详情页滚动到底部
2. 点击"归档订阅"按钮
3. 订阅被归档后自动返回列表页

### 删除订阅

1. 在详情页滚动到底部
2. 点击"删除订阅"按钮
3. 在确认对话框中点击"删除"
4. 订阅被删除后自动返回列表页

## 技术实现

### 状态管理

- 使用 `@State` 管理编辑模式状态
- 使用 `editedSubscription` 临时存储编辑中的数据
- 只有在保存时才更新原始订阅对象

### 导航

- 使用 `NavigationLink` 实现页面导航
- 使用 `.navigationBarBackButtonHidden(isEditMode)` 在编辑模式隐藏返回按钮
- 使用 `@Environment(\.dismiss)` 实现返回上一页

### 数据持久化

- 直接修改 SwiftData 模型对象
- 调用 `modelContext.save()` 保存更改
- 使用 ViewModel 的 service 方法处理归档和删除

### UI 组件

- 使用 `Form` 实现编辑界面
- 使用自定义 `InfoRow` 组件显示信息
- 使用 `infoCard` 辅助方法创建卡片布局

## 注意事项

1. **编辑模式导航**
   - 编辑模式下隐藏了系统返回按钮
   - 必须通过"取消"或"保存"按钮退出编辑模式

2. **数据同步**
   - 保存后会自动刷新详情页数据
   - 归档/删除后会自动返回列表页

3. **错误处理**
   - 所有操作都有 try-catch 错误处理
   - 错误会通过 Toast 消息显示给用户

4. **确认对话框**
   - 删除操作需要用户确认
   - 防止误操作
