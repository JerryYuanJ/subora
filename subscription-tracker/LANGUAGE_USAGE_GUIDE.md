# 语言切换使用指南

## 快速开始

### 切换应用语言

应用的语言跟随系统设置。要更改应用语言：

1. 打开应用
2. 点击底部的"设置"标签
3. 在"外观"部分找到"语言"选项
4. 点击"语言"行（会显示当前语言和一个跳转图标）
5. 系统会自动跳转到 iOS 系统设置页面
6. 在系统设置中更改语言：
   - 设置 → 通用 → 语言与地区 → 首选语言
7. 返回应用，语言已更新

## 功能特点

✅ **跟随系统** - 应用语言自动跟随 iOS 系统语言设置

✅ **标准体验** - 符合 iOS 标准的语言切换流程

✅ **全局生效** - 所有页面的文本都会更新，包括：

- 底部导航栏标签
- 设置页面
- 订阅列表
- 分类管理
- 洞察页面
- 所有按钮和提示信息

## 常见问题

### Q: 如何更改应用语言？

A: 点击设置中的"语言"选项，会跳转到系统设置页面。在系统设置中更改语言后，返回应用即可看到新语言。

### Q: 为什么不能在应用内直接切换语言？

A: 为了提供更好的用户体验和符合 iOS 标准，应用语言跟随系统设置。这样可以确保所有系统组件（如日期选择器、时间选择器等）也使用相同的语言。

### Q: 更改语言后需要重启应用吗？

A: 从系统设置返回应用后，语言会自动更新，无需手动重启。

### Q: 如何查看当前使用的语言？

A: 在设置页面的"语言"选项右侧会显示当前使用的语言。

## 技术说明

应用使用 iOS 标准的本地化机制：

- 所有界面文本存储在对应语言的 `Localizable.strings` 文件中
- 应用语言自动跟随系统语言设置
- 使用 `NSLocalizedString` 获取本地化字符串
- 确保翻译的准确性和一致性

## 开发者信息

如需添加新的本地化字符串：

1. 在 `LocalizationHelper.swift` 中添加新的键：

```swift
static let myNewKey = NSLocalizedString("my.new.key", comment: "Description")
```

2. 在所有语言的 `.strings` 文件中添加翻译：

```
// en.lproj/Localizable.strings
"my.new.key" = "English Text";

// zh-Hans.lproj/Localizable.strings
"my.new.key" = "中文文本";

// ja.lproj/Localizable.strings
"my.new.key" = "日本語テキスト";
```

3. 在视图中使用：

```swift
Text(L10n.MySection.myNewKey)
```
